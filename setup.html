What is TzeBot Substructure?

 TzeBot Substructure is a discord bot substructure which consists all of the existing and upcoming features of TzeBot, but not for using it, hosting your own bot. You are free to use it to host your own bot, you can also be a part of development of TzeBot Substructure.

Advanced Discord Bot Substructure

Automatic Saving & Loading

 TzeBot has a functionality that allows:

  • It has a database which consists languages and prefixes for each server that has these functionalities:
    1. Automatic Saving: TzeBot saves these values for every 15 minutes and even for unexpected error or shutdown.
    2. Automatic Loading: TzeBot loades these values for each startup.
  • It also has a version-control mechanism which allows you to:
    1. Version Control: See if it is a new version of TzeBot it checks version every startup.
    2. Update Button: It will be only visible when an update is available. You can just click on it to get latest version and change notes.

How can I use it as a substructure?

Using TzeBot Substructure For Your Bot

 In order to use TzeBot Substructure you have to set values listed below after downloading and launching TzeBot-x.xx.jar:

Download Latest Version
  1. Discord ID of Owner: You have to get Discord ID of Owner by just turning on developers stuff on discord settings.
  2. Default Prefix: We have to use prefix to call commands. Default is '.'.
  3. Discord Bot Token: You have to get your discord bot's token in discord developer. console.
  4. Youtube API Key: Just google it and create an application then get a key for your 'YouTube Data API v3' application.
  5. Shard(s): Except extraordinary situations you don't have to set it. It will automatically sets shard number according to your server number.

And just hit 'Start' button to start your own bot that using TzeBot's substructure.
